Requests

If users want access child app (such as nextcloud, rocketchat, ..) they need to access central app to request rights with admin.
Step 1: Users login to central app using aws account.
Step 2: Users first login to child app to register with it.
Step 3: Users click request button and wait for admin accept request rights, if users click Request button without first login to child app, a error popup will be showed.
Step 4: Admin accept request of a user to grant he or she permission to access child app. Admin can do this step in admin request page but also do in admin detail user page.
